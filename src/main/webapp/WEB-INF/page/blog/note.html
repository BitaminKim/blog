    用cglib生成的代理类取不到注解的问题

    直接在注解上面加@Inherited CGLIB 返回的代理类是基于原来类的子类的，加上这个标志就可以令子类继承这个注解

    eclipse tomcat 启动45秒超时问题

    找到workplace路径/.metadata/.plugins /org.eclipse.wst.server.core/servers.xml， 编辑start-timeout，把45改大点就好。
